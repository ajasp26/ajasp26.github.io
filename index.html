Certification Courses

TCM's Practical Ethical Hacker Course

As part of my PNPT/OSCP studies, I completed TCM's Practical Ethical Hacker course as part of my PNPT (Practical Network Penetration Tester) preparation. This course provides hands-on training focused on real-world tools and techniques used by professional ethical hackers. It covers essential topics such as network penetration testing, web application security, wireless attacks, and Active Directory exploitation. Students gain practical skills in Linux, Python, and note-keeping while also learning key concepts like reconnaissance, exploitation, and post-exploitation techniques. The course includes a mid-course capstone where there are several machine instances, ranging in difficulty, which they use various tools and techniques to compromise them. Students set up and then compromise an Active Directory along with performing web application penetration testing gaining experience with exploiting OWASP Top 10 vulnerabilities and others.

TryHackMe Junior Penetration Tester

This comprehensive learning path covers essential topics such as network scanning, enumeration, exploitation, privilege escalation, and web application security. Through hands-on labs and practical exercises, students gain experience with key tools like Metasploit, Burp Suite, and Nmap, while also learning how to document findings and generate professional reports. By the end of the path, learners will have a strong understanding of real-world penetration testing techniques and methodologies, preparing them for junior roles in cybersecurity.
